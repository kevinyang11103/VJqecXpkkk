## 项目概述


## 技术栈

- 前端：HTML, CSS, JavaScript
- 后端：Python, Django
- 数据库：MySQL

## 功能模块

- 用户管理
  - 用户注册与登录
  - 权限控制
- 商品管理
  - 商品信息维护
  - 库存管理
- 订单管理
  - 购物车管理
  - 订单创建与处理
  - 订单查询
- 报告与统计
  - 销售报告
  - 库存报告

## 系统角色

- 系统管理员：负责用户管理、系统设置等
- 超市员工：负责商品管理、订单处理等
- 普通用户：进行商品浏览、购买等操作

## 运行环境

- Python 3.8+
- Django 3.2+
- MySQL 5.7+
- 操作系统：兼容Windows、Linux、MacOS

## 部署步骤

1. 环境准备

2. 数据库部署
   - 创建数据库及用户
   - 导入数据库结构及数据
3. 应用部署

   - 配置应用设置
   - 收集静态文件
4. 运行
   - 运行Django开发服务器
